# Zepto Inventory Data Analysis using PostgreSQL

This project focuses on analyzing Zepto’s inventory dataset using PostgreSQL to generate meaningful business insights and support data-driven decision-making.

The analysis includes data cleaning, exploratory data analysis (EDA), business insight generation, and advanced SQL techniques such as window functions.

Key objectives of this project:

• Clean and prepare raw inventory data for analysis
• Analyze product pricing, discounts, and inventory distribution
• Identify top revenue-generating products and categories
• Detect high-demand and out-of-stock products
• Perform product ranking and segmentation using window functions
• Generate actionable business insights

Skills and Tools Used:

• PostgreSQL
• SQL
• Data Cleaning
• Window Functions
• Business Analysis
• Data Exploration

Key Business Insights:

• Identified categories contributing the highest revenue potential
• Found premium and high-margin products
• Analyzed discount strategies across categories
• Performed Pareto analysis to identify key revenue drivers

This project demonstrates strong SQL, analytical, and business problem-solving skills relevant for Data Analyst and Business Analyst roles.
